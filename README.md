
# Cephalo // SketchDrop

Next.js 14 (App Router) + Prisma + PostgreSQL + Supabase Storage + Vipps (stub endpoints) + Email invites + Forgot/Reset password.

## Quick start

```bash
pnpm install
cp .env.example .env    # fill values
pnpm prisma:migrate
pnpm prisma:seed
pnpm dev
```

Open http://localhost:3000/login
